# Réplica de Uber Eats (Proyecto universitario, 2022)
Descripción: Réplica de las funcionalidades principales de Uber Eats, incluyendo la experiencia del cliente y la del repartidor. El proyecto se desarrolló sin backend, utilizando JavaScript, PHP, JSON, y almacenamiento local para la gestión de datos.
# Tecnologías: 
JavaScript, PHP, JSON, LocalStorage

Responsabilidades: Diseño de la interfaz y desarrollo de la lógica para la gestión de repartidores.

Repositorio: https://github.com/rick2303/CREW
